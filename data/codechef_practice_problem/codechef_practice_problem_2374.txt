A Dr.D's Legacy OriginalA Dr.D's Legacy Original
Dr.D wishes to choose exactly KK distinct integers from a given interval [L,R][L,R] in such a way that the bitwise AND of the chosen integers is maximum possible. Find this maximum value.
Input

The first line of the input contains a single integer TT denoting the number of test cases. The description of TT test cases follows.
The first and only line of each test case contains three space-separated integers LL, RR and KK.

Output
For each test case, print a single line containing one integer — the maximum value of the bitwise AND.
Constraints

1≤T≤1061≤T≤106
0≤L≤R≤10180≤L≤R≤1018
1≤K≤R−L+11≤K≤R−L+1


                        Sample Input 1
                        





6
1 9 3
4 7 1
10 75 12
3 8 3
5 10 2
192 913893 3812



                        Sample Output 1
                        





4
7
64
4
8
909312
